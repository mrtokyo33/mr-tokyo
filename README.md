# O que é o projeto? 
Este projeto será um site voltado para meu portfólio e publicação de artigos.

A primeira funcionalidade será um painel que exibe meus artigos, permitindo com que eles sejam organizados e visualizados de forma simples e estética.

No entanto, o site não será limitado apenas a artigos. O conteúdo será dividido por áreas, de modo que, sempre que eu quiser ensinar ou mostrar algo, poderei criar um arquivo e publicá-lo. O próprio site será responsável por formatar e exibir o conteúdo automaticamente.

No futuro, o projeto será expandido para incluir lições, sistema de login, progressão por XP e outras funcionalidades voltadas à experiência do usuário.
